# movie list app

Made with MERN stack: mongodb, express, react, nodejs, material UI, jest, axios.

## backend/.env

Rename .env.example to .env.\
Add your OMDB_API_KEY in .env.\
<https://www.omdbapi.com>

## requirements

mongodb\
node

## setup

```cmd
cd movie-app
npm i
npm run backend
cd frontend
npm i
cd ..
npm run frontend
```
